# Affectations simples, parallèles et multiples 

![Facile](https://img.shields.io/badge/-Facile-success)

## Énoncé

Dans cet exercice, vous allez devoir réaliser plusieurs affectations de variables.

Voici les 6 opérations que vous devez réaliser :

- Créez une variable `a` qui a pour valeur le nombre entier 500.
- Créez une variable `langage` qui a pour valeur la chaîne de caractères 'python'.
- Assignez à une variable `b` la même valeur que celle contenue dans la variable a (pas le droit de faire `b = 500` !).
- Créez, avec une seule ligne de code, deux variables `c` et `d`, qui contiennent toutes deux la valeur 600.
- Créez, avec une seule ligne de code, deux variables `e` et `f`, qui contiennent respectivement les valeurs 8 et 12.
- Maintenant, avec une seule ligne de code, permutez les valeurs contenues dans `e` et `f` (`f` doit être égal à 8 et `e` à 12)
